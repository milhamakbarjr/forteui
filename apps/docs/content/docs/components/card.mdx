# Card

Card component for grouping related content in a visually distinct container. Includes composable header, content, and footer sections with flexible styling options.

## Installation

```bash
npm install forteui-core
```

## Import

```tsx
import { Card, CardHeader, CardContent, CardFooter, CardTitle, CardDescription } from 'forteui-core';
```

## Basic Usage

```tsx
import { Card, CardHeader, CardContent, CardFooter, CardTitle, CardDescription } from 'forteui-core';

function MyComponent() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
        <CardDescription>Card description goes here</CardDescription>
      </CardHeader>
      <CardContent>
        <p>This is the card content area.</p>
      </CardContent>
      <CardFooter>
        <Button>Action</Button>
      </CardFooter>
    </Card>
  );
}
```

## Card Variants

Cards support different visual styles for various use cases:

```tsx
{/* Default card with subtle shadow */}
<Card>
  <CardContent>Default card content</CardContent>
</Card>

{/* Elevated card with enhanced shadow */}
<Card variant="elevated">
  <CardContent>Elevated card content</CardContent>
</Card>

{/* Outlined card with border, no shadow */}
<Card variant="outlined">
  <CardContent>Outlined card content</CardContent>
</Card>

{/* Filled card with background color */}
<Card variant="filled">
  <CardContent>Filled card content</CardContent>
</Card>
```

## Padding Options

Control the spacing within cards using the padding prop:

```tsx
{/* Compact spacing (16px) */}
<Card padding="sm">
  <CardHeader padding="sm">
    <CardTitle>Small Padding</CardTitle>
  </CardHeader>
  <CardContent padding="sm">
    Compact content
  </CardContent>
</Card>

{/* Default spacing (24px) */}
<Card padding="md">
  <CardHeader padding="md">
    <CardTitle>Medium Padding</CardTitle>
  </CardHeader>
  <CardContent padding="md">
    Standard content
  </CardContent>
</Card>

{/* Spacious layout (32px) */}
<Card padding="lg">
  <CardHeader padding="lg">
    <CardTitle>Large Padding</CardTitle>
  </CardHeader>
  <CardContent padding="lg">
    Expanded content
  </CardContent>
</Card>
```

## Composition Patterns

### Full Card Structure

```tsx
<Card>
  <CardHeader>
    <CardTitle>Project Update</CardTitle>
    <CardDescription>2 hours ago</CardDescription>
  </CardHeader>
  <CardContent>
    <p>The new features have been successfully deployed to production.</p>
  </CardContent>
  <CardFooter>
    <Button variant="primary">View Details</Button>
    <Button variant="outline-default">Dismiss</Button>
  </CardFooter>
</Card>
```

### Content-Only Card

```tsx
<Card>
  <CardContent>
    <div className="text-center py-8">
      <h2 className="text-2xl font-bold mb-4">Welcome!</h2>
      <p className="text-gray-600 mb-6">Get started with your dashboard.</p>
      <Button variant="primary">Get Started</Button>
    </div>
  </CardContent>
</Card>
```

## Interactive Cards

### Clickable Card

```tsx
<Card 
  className="cursor-pointer hover:shadow-lg transition-shadow"
  onClick={() => console.log('Card clicked')}
>
  <CardContent>
    <div className="flex items-center space-x-4">
      <div className="w-12 h-12 bg-primary-100 rounded-lg flex items-center justify-center">
        <IconFolder className="w-6 h-6 text-primary-600" />
      </div>
      <div>
        <h3 className="font-semibold">Documents</h3>
        <p className="text-sm text-gray-500">24 files</p>
      </div>
    </div>
  </CardContent>
</Card>
```

### Form Integration

```tsx
<Card>
  <CardHeader>
    <CardTitle>Contact Form</CardTitle>
    <CardDescription>Send us a message</CardDescription>
  </CardHeader>
  <CardContent className="space-y-4">
    <Input label="Name" placeholder="Enter your name" />
    <Input label="Email" placeholder="Enter your email" />
    <Textarea
      label="Message"
      placeholder="How can we help you?"
      rows={3}
    />
  </CardContent>
  <CardFooter className="flex justify-end gap-3">
    <Button variant="outline-default">Cancel</Button>
    <Button variant="primary">Submit</Button>
  </CardFooter>
</Card>
```
