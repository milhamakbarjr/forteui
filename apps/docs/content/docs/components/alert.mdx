# Alert

Alert component for displaying important messages and notifications. Supports multiple severity levels, variants, and optional close functionality with excellent accessibility.

## Installation

```bash
npm install forteui-core
```

## Import

```tsx
import { Alert } from 'forteui-core';
```

## Basic Usage

```tsx
import { Alert } from 'forteui-core';

function MyComponent() {
  return (
    <Alert severity="info">
      This is an informational alert.
    </Alert>
  );
}
```

## Severity Levels

Alerts support four semantic severity levels:

- **Info**: For informational messages (blue theme)
- **Success**: For positive feedback (green theme)  
- **Warning**: For caution messages (yellow theme)
- **Error**: For error notifications (red theme)

## Variants

### Default Variant
Light background with subtle border styling.

```tsx
<Alert variant="default" severity="info">
  Default variant with subtle styling
</Alert>
```

### Filled Variant
Solid background with strong semantic colors.

```tsx
<Alert variant="filled" severity="success">
  Filled variant with solid background
</Alert>
```

### Outline Variant
Emphasized border with light background.

```tsx
<Alert variant="outline" severity="warning">
  Outline variant with border emphasis
</Alert>
```

## Interactive Features

### Closable Alerts
Add a close button to allow users to dismiss alerts:

```tsx
<Alert 
  severity="info" 
  closable 
  onClose={() => console.log('Alert closed')}
>
  This alert can be closed
</Alert>
```

### Custom Icons
Replace the default severity icon with a custom one:

```tsx
import { Alert, IconSettings } from 'forteui-core';

<Alert severity="info" icon={(props) => <IconSettings {...props} />}>
  Alert with custom icon
</Alert>
```

### Hide Icon
Remove the icon completely for a cleaner look:

```tsx
<Alert severity="info" hideIcon>
  Alert without an icon
</Alert>
```
