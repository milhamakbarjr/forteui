# Avatar

Avatar component for displaying user profile images with intelligent fallback handling, multiple sizes, and optional status indicators. Perfect for user profiles, comments, and team displays.

## Installation

```bash
npm install @forteui/core
```

## Import

```tsx
import { Avatar } from '@forteui/core';
```

## Basic Usage

```tsx
import { Avatar } from '@forteui/core';

function MyComponent() {
  return (
    <Avatar 
      src="/user-avatar.jpg"
      alt="John Doe"
      fallback="JD"
    />
  );
}
```

## Interactive Avatars

### Clickable Avatar

```tsx
<Avatar 
  src="/user.jpg"
  fallback="JD"
  className="cursor-pointer hover:ring-2 hover:ring-primary-main transition-all"
  onClick={() => console.log('Avatar clicked')}
/>
```

### Avatar with Dropdown

```tsx
import { useState } from 'react';

function AvatarWithMenu() {
  const [isOpen, setIsOpen] = useState(false);
  
  return (
    <div className="relative">
      <Avatar 
        src="/user.jpg"
        fallback="JD"
        className="cursor-pointer"
        onClick={() => setIsOpen(!isOpen)}
      />
      {isOpen && (
        <div className="absolute top-12 right-0 bg-white border rounded-lg shadow-lg p-2 min-w-[150px]">
          <button className="w-full text-left px-3 py-2 hover:bg-gray-100 rounded">Profile</button>
          <button className="w-full text-left px-3 py-2 hover:bg-gray-100 rounded">Settings</button>
          <button className="w-full text-left px-3 py-2 hover:bg-gray-100 rounded">Sign Out</button>
        </div>
      )}
    </div>
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | - | Image source URL |
| `alt` | `string` | - | Alt text for the image |
| `fallback` | `string \| ReactNode` | - | Fallback content when image fails to load |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl'` | `'md'` | Size of the avatar |
| `status` | `'online' \| 'away' \| 'busy' \| 'offline'` | - | Status indicator |
| `className` | `string` | - | Additional CSS classes |
| `onClick` | `() => void` | - | Click handler for interactive avatars |

## Design Specifications

- **Sizes**: XS (24px), SM (32px), MD (40px), LG (48px), XL (64px), 2XL (80px)
- **Shape**: Perfect circle
- **Status Indicator**: 8px circle positioned at bottom-right
- **Fallback**: Centered text or icon with background color
- **Border**: Optional border for layered avatars

## Accessibility

- **Alt Text**: Properly describes the user when image is present
- **Keyboard Navigation**: Supports Tab navigation when interactive
- **Screen Readers**: Fallback content is announced appropriately
- **Focus Management**: Clear focus indicators for interactive avatars

## Best Practices

1. **Consistent sizing**: Use the same size for avatars in the same context
2. **Meaningful fallbacks**: Use initials or relevant icons
3. **Alt text**: Provide descriptive alt text for images
4. **Status clarity**: Use status indicators consistently across the app
5. **Loading states**: Consider loading placeholders for slow images
