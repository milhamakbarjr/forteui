# Avatar

Avatar component for displaying user profile images with intelligent fallback handling, multiple sizes, shapes, variants, and optional status indicators. Perfect for user profiles, comments, and team displays.

## Installation

```bash
npm install forteui-core
```

## Import

```tsx
import { Avatar } from 'forteui-core';
```

## Basic Usage

```tsx
import { Avatar } from 'forteui-core';

function MyComponent() {
  return (
    <Avatar 
      src="/user-avatar.jpg"
      alt="John Doe"
      fallback="JD"
    />
  );
}
```

## Sizes

Available sizes: `xs` (24px), `sm` (32px), `md` (40px), `lg` (48px), `xl` (64px)

```tsx
<div className="flex items-center gap-4">
  <Avatar size="xs" src="/user.jpg" fallback="XS" />
  <Avatar size="sm" src="/user.jpg" fallback="SM" />
  <Avatar size="md" src="/user.jpg" fallback="MD" />
  <Avatar size="lg" src="/user.jpg" fallback="LG" />
  <Avatar size="xl" src="/user.jpg" fallback="XL" />
</div>
```

## Variants

Avatar supports semantic color variants:

```tsx
<div className="flex items-center gap-4">
  <Avatar variant="default" fallback="DE" />
  <Avatar variant="primary" fallback="PR" />
  <Avatar variant="secondary" fallback="SE" />
  <Avatar variant="info" fallback="IN" />
  <Avatar variant="success" fallback="SU" />
  <Avatar variant="warning" fallback="WA" />
  <Avatar variant="error" fallback="ER" />
</div>
```

## Shapes

Choose from circular, rounded or square shapes:

```tsx
<div className="flex items-center gap-4">

  <Avatar shape="rounded" variant="primary" fallback="RO" />

```

## Interactive Avatars

### Clickable Avatar

```tsx
<Avatar 
  src="/user.jpg"
  fallback="JD"
  className="cursor-pointer hover:ring-2 hover:ring-primary-main transition-all"
  onClick={() => console.log('Avatar clicked')}
/>
```

### Avatar with Dropdown

```tsx
import { useState } from 'react';

function AvatarWithMenu() {
  const [isOpen, setIsOpen] = useState(false);
  
  return (
    <div className="relative">
      <Avatar 
        src="/user.jpg"
        fallback="JD"
        className="cursor-pointer"
        onClick={() => setIsOpen(!isOpen)}
      />
      {isOpen && (
        <div className="absolute top-12 right-0 bg-white border rounded-lg shadow-lg p-2 min-w-[150px]">
          <button className="w-full text-left px-3 py-2 hover:bg-gray-100 rounded">Profile</button>
          <button className="w-full text-left px-3 py-2 hover:bg-gray-100 rounded">Settings</button>
          <button className="w-full text-left px-3 py-2 hover:bg-gray-100 rounded">Sign Out</button>
        </div>
      )}
    </div>
  );
}
```
