# Button

A versatile button component that supports multiple variants, sizes, states, and rich icon integration. Built with accessibility in mind.

## Installation

```bash
npm install forteui-core
```

## Import

```tsx
import { Button } from 'forteui-core';
```

## Basic Usage

The simplest way to use the Button component:

```tsx
import { Button } from 'forteui-core';

function MyComponent() {
  return (
    <Button variant="primary" onClick={() => alert('Clicked!')}>
      Click me
    </Button>
  );
}
```

## Variants

### Contained Buttons (Default)

Contained buttons are high-emphasis buttons that are distinguished by their use of elevation and fill.

```tsx
<Button variant="default">Default</Button>
<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="info">Info</Button>
<Button variant="success">Success</Button>
<Button variant="warning">Warning</Button>
<Button variant="error">Error</Button>
```

### Outline Buttons

Outline buttons are medium-emphasis buttons that provide an alternative to contained buttons.

```tsx
<Button variant="outline-default">Default</Button>
<Button variant="outline-primary">Primary</Button>
<Button variant="outline-info">Info</Button>
<Button variant="outline-success">Success</Button>
<Button variant="outline-warning">Warning</Button>
<Button variant="outline-error">Error</Button>
```

### Text Buttons

Text buttons are typically used for less-pronounced actions, including those located in cards and dialogs.

```tsx
<Button variant="text-default">Default</Button>
<Button variant="text-primary">Primary</Button>
<Button variant="text-info">Info</Button>
<Button variant="text-success">Success</Button>
<Button variant="text-warning">Warning</Button>
<Button variant="text-error">Error</Button>
```

### Soft Buttons

Soft buttons provide a subtle background color while maintaining good contrast.

```tsx
<Button variant="soft-default">Default</Button>
<Button variant="soft-primary">Primary</Button>
<Button variant="soft-info">Info</Button>
<Button variant="soft-success">Success</Button>
<Button variant="soft-warning">Warning</Button>
<Button variant="soft-error">Error</Button>
```

### Icon Buttons

Icon buttons are circular buttons designed specifically for icon-only actions. They provide hover backgrounds and are perfect for toolbars and compact interfaces.

```tsx
import { Button, IconHome, IconUser, IconPlus, IconX } from 'forteui-core';

// Different icon button variants
<Button variant="icon-default" size="icon-md"><IconHome size={20} /></Button>
<Button variant="icon-primary" size="icon-md"><IconUser size={20} /></Button>
<Button variant="icon-info" size="icon-md"><IconPlus size={20} /></Button>
<Button variant="icon-error" size="icon-md"><IconX size={20} /></Button>
```

#### Icon Button Sizes

Icon buttons come in four different sizes:

```tsx
<Button variant="icon-primary" size="icon-xs"><IconHome size={16} /></Button>
<Button variant="icon-primary" size="icon-sm"><IconHome size={18} /></Button>
<Button variant="icon-primary" size="icon-md"><IconHome size={20} /></Button>
<Button variant="icon-primary" size="icon-lg"><IconHome size={24} /></Button>
```

## Sizes

The Button component comes in three sizes to fit various design needs:

```tsx
<Button variant="primary" size="sm">Small</Button>
<Button variant="primary" size="md">Medium</Button>
<Button variant="primary" size="lg">Large</Button>
```

## States

### Disabled State

Buttons can be disabled using the `disabled` prop:

```tsx
<Button variant="primary" disabled>Disabled Button</Button>
<Button variant="outline-primary" disabled>Disabled Outline</Button>
<Button variant="text-primary" disabled>Disabled Text</Button>
<Button variant="soft-primary" disabled>Disabled Soft</Button>
```

## Icons

ForteUI includes Tabler Icons out of the box. You can import icons directly from the core package.

### Import Icons

```tsx
import { Button, IconPlus, IconCheck, IconTrash, IconHome, IconUser } from 'forteui-core';
```

### Buttons with Leading Icons

Add icons before the button text:

```tsx
<Button variant="primary">
  <IconPlus size={16} />
  <span>Add Item</span>
</Button>

<Button variant="success">
  <IconCheck size={16} />
  <span>Confirm</span>
</Button>

<Button variant="error">
  <IconTrash size={16} />
  <span>Delete</span>
</Button>
```

### Buttons with Trailing Icons

Add icons after the button text:

```tsx
import { Button, IconArrowRight, IconChevronDown, IconDownload } from 'forteui-core';

<Button variant="primary">
  <span>Next</span>
  <IconArrowRight size={16} />
</Button>

<Button variant="secondary">
  <span>Options</span>
  <IconChevronDown size={16} />
</Button>

<Button variant="success">
  <span>Download</span>
  <IconDownload size={16} />
</Button>
```

### Icon-Only Buttons

For icon-only buttons, use the icon variants:

```tsx
import { Button, IconHome, IconUser, IconSettings } from 'forteui-core';

<Button variant="icon-default" size="icon-md">
  <IconHome className="w-6 h-6" />
</Button>

<Button variant="icon-primary" size="icon-md">
  <IconUser className="w-6 h-6" />
</Button>

<Button variant="icon-info" size="icon-md">
  <IconSettings className="w-6 h-6" />
</Button>
```

#### Icon Button Sizes

Icon buttons come in four sizes:

```tsx
<Button variant="icon-primary" size="icon-lg">
  <IconPlus className="w-8 h-8" />
</Button>
<Button variant="icon-primary" size="icon-md">
  <IconPlus className="w-6 h-6" />
</Button>
<Button variant="icon-primary" size="icon-sm">
  <IconPlus className="w-5 h-5" />
</Button>
<Button variant="icon-primary" size="icon-xs">
  <IconPlus className="w-4 h-4" />
</Button>
```

## Advanced Usage

### As Child (Polymorphic)

Use the `asChild` prop to render the button as a different element:

```tsx
import { Button } from 'forteui-core';
import Link from 'next/link';

<Button asChild>
  <Link href="/dashboard">Go to Dashboard</Link>
</Button>
```

### Custom Click Handlers

```tsx
const handleClick = () => {
  console.log('Button clicked!');
  // Your custom logic here
};

<Button variant="primary" onClick={handleClick}>
  Custom Handler
</Button>
```
