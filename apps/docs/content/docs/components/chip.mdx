# Chip

Compact elements that represent inputs, attributes, or actions. Chips allow users to enter information, make selections, filter content, or trigger actions.

## Installation

```bash
npm install @forteui/core
```

## Removable Chips

### Basic Removable

```tsx
import { useState } from 'react';
import { Chip, IconX } from '@forteui/core';

function RemovableChipsExample() {
  const [chips, setChips] = useState([
    { id: 1, label: 'React' },
    { id: 2, label: 'TypeScript' },
    { id: 3, label: 'Tailwind CSS' },
    { id: 4, label: 'Next.js' }
  ]);
  
  const handleDelete = (chipId: number) => {
    setChips(chips.filter(chip => chip.id !== chipId));
  };
  
  return (
    <div className="flex flex-wrap gap-2">
      {chips.map(chip => (
        <Chip 
          key={chip.id}
          variant="filled"
          color="primary"
          size="medium"
          endIcon={<IconX />}
          onEndIconClick={() => handleDelete(chip.id)}
        >
          {chip.label}
        </Chip>
      ))}
    </div>
  );
}
```

## API Reference

### Props

The Chip component accepts the following props:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'filled' \| 'outlined' \| 'soft'` | `'filled'` | The visual style variant of the chip |
| `color` | `'default' \| 'primary' \| 'secondary' \| 'info' \| 'success' \| 'warning' \| 'error'` | `'default'` | The color theme of the chip |
| `size` | `'small' \| 'medium'` | `'medium'` | The size of the chip |
| `startIcon` | `React.ReactNode` | - | Icon to display at the start (left side) of the chip |
| `endIcon` | `React.ReactNode` | - | Icon to display at the end (right side) of the chip |
| `onStartIconClick` | `() => void` | - | Handler function called when the start icon is clicked |
| `onEndIconClick` | `() => void` | - | Handler function called when the end icon is clicked |
| `children` | `React.ReactNode` | - | The content to display inside the chip |
| `className` | `string` | - | Additional CSS classes to apply |

#### Interactive Example

```tsx
import { useState } from 'react';
import { Chip, IconX } from '@forteui/core';

function InteractiveChips() {
  const [tags, setTags] = useState(['React', 'TypeScript', 'Next.js']);
  
  const removeTag = (tagToRemove: string) => {
    setTags(tags.filter(tag => tag !== tagToRemove));
  };
  
  return (
    <div className="flex flex-wrap gap-2">
      {tags.map(tag => (
        <Chip 
          key={tag}
          variant="filled"
          color="primary"
          size="medium"
          endIcon={<IconX />}
          onEndIconClick={() => removeTag(tag)}
        >
          {tag}
        </Chip>
      ))}
    </div>
  );
}
```

## Design Specifications

The Chip component follows exact design specifications:

### Medium Size (32px height)
- Height: 32px (fixed)
- Outer padding: 4px left/right
- Label padding: 8px left/right
- Border radius: 10px (filled), 8px (outlined/soft)
- Typography: 13px, weight 500, line-height 18px

### Small Size (24px height)
- Height: 24px (fixed)
- Outer padding: 3px left/right
- Label padding: 5px left/right
- Border radius: 8px (all variants)
- Typography: 13px, weight 500, line-height 18px

### Action Icons
- Icon container: 20px Ã— 20px (both sizes)
- Icon size: 16.67px
- Opacity: 48% (default), 75% (hover)
- Start icon margin-right: 4px
- End icon margin-left: 4px

### Color Variants

- **Filled**: Background uses semantic color main, text uses contrast colors
- **Outlined**: 1px solid border with semantic color, transparent background
- **Soft**: 16% alpha semantic color background, darker text variant
