# Badge

Badge component for displaying status indicators, counts, and labels. Perfect for notification badges, status indicators, and categorical labels.

## Installation

```bash
npm install forteui-core
```

## Import

```tsx
import { Badge } from 'forteui-core';
```

## Basic Usage

```tsx
import { Badge } from 'forteui-core';

function MyComponent() {
  return (
    <Badge variant="filled" color="primary">
      New
    </Badge>
  );
}
```

## Common Use Cases

### Notification Count

```tsx
<div className="relative">
  <Button variant="outline-primary">
    Messages
  </Button>
  <Badge 
    variant="filled" 
    color="error" 
    size="sm"
    className="absolute -top-2 -right-2"
  >
    3
  </Badge>
</div>
```

### Status Indicators

```tsx
<div className="flex items-center gap-2">
  <span>Server Status:</span>
  <Badge variant="filled" color="success">Online</Badge>
</div>

<div className="flex items-center gap-2">
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'filled' \| 'outline' \| 'soft'` | `'filled'` | Visual style variant |
| `color` | `'default' \| 'primary' \| 'secondary' \| 'success' \| 'warning' \| 'error'` | `'default'` | Color theme |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size of the badge |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Badge content |

## Design Specifications

- **Border Radius**: Fully rounded (pill shape)
- **Typography**: Bold weight for emphasis
- **Padding**: Proportional to size
- **Colors**: Uses semantic color system
- **Sizing**: Small (20px), Medium (24px), Large (28px) height

## Accessibility

- **Color Independence**: Information is not conveyed by color alone
- **Readable Text**: High contrast ratios maintained
- **Screen Readers**: Content is properly announced

## Best Practices

1. **Keep text short**: Use concise labels or numbers
2. **Consistent sizing**: Use appropriate size for context
3. **Semantic colors**: Use colors that match the meaning
4. **Strategic placement**: Position near related content
5. **Don't overuse**: Avoid too many badges in one area
