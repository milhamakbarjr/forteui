# Badge

Badge wrapper component for displaying count numbers or dot indicators on top of child elements. Perfect for notification badges on icons, unread message counts, and status indicators.

## Installation

```bash
npm install forteui-core
```

## Import

```tsx
import { Badge } from 'forteui-core';
```

## Basic Usage

```tsx
import { Badge } from 'forteui-core';
import { IconMail } from 'forteui-core';

function MyComponent() {
  return (
    <Badge variant="primary" count={5}>
      <IconMail size={24} />
    </Badge>
  );
}
```

## Interactive Examples

### Real-time Counter

```tsx
import { useState } from 'react';
import { Badge } from 'forteui-core';
import { IconMail, IconShoppingCart } from 'forteui-core';

function InteractiveExample() {
  const [messageCount, setMessageCount] = useState(3);
  const [cartCount, setCartCount] = useState(0);
  
  return (
    <div className="flex gap-6">
      <div className="flex flex-col items-center gap-2">
        <Badge variant="primary" count={messageCount}>
          <IconMail size={32} />
        </Badge>
        <button 
          onClick={() => setMessageCount(prev => prev + 1)}
          className="px-3 py-1 bg-blue-100 rounded text-sm"
        >
          Add Message ({messageCount})
        </button>
      </div>
      
      <div className="flex flex-col items-center gap-2">
        <Badge variant="success" count={cartCount} showZero>
          <IconShoppingCart size={32} />
        </Badge>
        <button 
          onClick={() => setCartCount(prev => prev + 1)}
          className="px-3 py-1 bg-green-100 rounded text-sm"
        >
          Add to Cart ({cartCount})
        </button>
      </div>
    </div>
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Child element to wrap with badge |
| `count` | `number` | `0` | Number to display in badge |
| `showZero` | `boolean` | `false` | Whether to show badge when count is 0 |
| `max` | `number` | `99` | Maximum number before showing "max+" |
| `type` | `'number' \| 'dot'` | `'number'` | Badge display type |
| `variant` | `'default' \| 'primary' \| 'secondary' \| 'info' \| 'success' \| 'warning' \| 'error'` | `'default'` | Badge color variant |
| `className` | `string` | - | Additional CSS classes |

## Design Specifications

### Positioning
- **Number badges**: Positioned at `-top-2` with dynamic right offset
- **Dot badges**: Positioned at `-top-1.5, -right-1.5`
- **Border**: 2px white border for separation from background

### Sizing
- **Number badges**: Minimum 20px height/width, 6px horizontal padding
- **Dot badges**: 10px width/height (w-2.5 h-2.5)
- **Typography**: 12px font size with bold weight

### Overflow Behavior
- **Default max**: 99 (shows "99+" for counts â‰¥ 100)
- **Dynamic positioning**: Adjusts based on formatted count length
- **Customizable**: Use `max` prop for different overflow points

## Accessibility

- **Wrapper Component**: Badge acts as a wrapper around child elements
- **Semantic Colors**: Uses semantic color variants for meaning
- **Screen Readers**: Child content remains accessible
- **Focus Management**: Preserves focus behavior of wrapped elements

## Best Practices

1. **Use with Icons**: Pair with ForteUI icons for consistent appearance
2. **Semantic Colors**: Choose variant colors that match the badge purpose
3. **Reasonable Counts**: Use overflow handling for large numbers
4. **Consistent Sizing**: Use appropriate icon sizes (16px, 24px, 32px)
5. **Interactive Elements**: Badge works well with buttons and clickable icons
