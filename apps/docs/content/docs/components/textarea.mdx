# Textarea

A versatile multi-line text input component for collecting longer text content. Features auto-resize functionality, character counting, validation states, and seamless form integration.

## Installation

```bash
npm install forteui-core
```

## Import

```tsx
import { Textarea } from 'forteui-core';
```

## Basic Usage

```tsx
import { Textarea } from 'forteui-core';

function MyComponent() {
  return (
    <Textarea 
      placeholder="Enter your message..."
      rows={4}
    />
  );
}
```

## Character Counting

### With Character Limit

```tsx
import { useState } from 'react';

function CharacterLimitExample() {
  const [value, setValue] = useState('');
  const maxLength = 280;
  
  return (
    <Textarea 
      label="Tweet draft"
      placeholder="What's happening?"
      value={value}
      onChange={(e) => setValue(e.target.value)}
      maxLength={maxLength}
      showCharacterCount
      helperText={`${value.length}/${maxLength} characters`}
      state={value.length > maxLength * 0.9 ? 'warning' : 'default'}
      rows={3}
    />
  );
}
```

## Form Integration

### Controlled Component

```tsx
import { useState } from 'react';

function ControlledTextarea() {
  const [message, setMessage] = useState('');
  const [isSubmitting, setIsSubmitting] = useState(false);
  
  const handleSubmit = async (e) => {
    e.preventDefault();
    setIsSubmitting(true);
    // Simulate API call
    await new Promise(resolve => setTimeout(resolve, 1000));
    alert('Message sent!');
    setMessage('');
    setIsSubmitting(false);
  };
  
  return (
    <form onSubmit={handleSubmit} className="space-y-4">
      <Textarea 
        label="Your message"
        placeholder="Type your message here..."
        value={message}
        onChange={(e) => setMessage(e.target.value)}
        required
        minLength={10}
        maxLength={500}
        rows={4}
        helperText={`${message.length}/500 characters (minimum 10)`}
      />
      
      <Button 
        type="submit" 
        disabled={message.length < 10 || isSubmitting}
        variant="primary"
      >
        {isSubmitting ? 'Sending...' : 'Send Message'}
      </Button>
    </form>
  );
}
```

### React Hook Form Integration

```tsx
import { useForm } from 'react-hook-form';

function ContactForm() {
  const { register, handleSubmit, watch, formState: { errors } } = useForm();
  const messageValue = watch('message', '');
  
  const onSubmit = (data) => {
    console.log('Form data:', data);
  };
  
  return (
    <form onSubmit={handleSubmit(onSubmit)} className="space-y-4">
      <Textarea 
        label="Message"
        placeholder="Enter your message..."
        {...register('message', { 
          required: 'Message is required',
          minLength: { value: 20, message: 'Message must be at least 20 characters' },
          maxLength: { value: 1000, message: 'Message must not exceed 1000 characters' }
        })}
        state={errors.message ? 'error' : 'default'}
        helperText={errors.message?.message || `${messageValue.length}/1000 characters`}
        rows={5}
      />
      
      <Button type="submit" variant="primary">
        Submit
      </Button>
    </form>
  );
}
```

## Real-World Examples

### Comment System

```tsx
import { useState } from 'react';

function CommentBox({ onSubmit }) {
  const [comment, setComment] = useState('');
  const [isSubmitting, setIsSubmitting] = useState(false);
  
  const handleSubmit = async () => {
    if (comment.trim().length < 3) return;
    
    setIsSubmitting(true);
    await onSubmit(comment);
    setComment('');
    setIsSubmitting(false);
  };
  
  return (
    <Card>
      <CardContent className="p-4 space-y-4">
        <div className="flex items-center gap-3">
          <Avatar size="sm" />
          <div>
            <p className="font-medium">Add a comment</p>
            <p className="text-sm text-gray-500">Share your thoughts</p>
          </div>
        </div>
        
        <Textarea 
          placeholder="Write a comment..."
          value={comment}
          onChange={(e) => setComment(e.target.value)}
          autoResize
          minRows={2}
          maxRows={6}
        />
        
        <div className="flex justify-between items-center">
          <span className="text-sm text-gray-500">
            {comment.length} characters
          </span>
          
          <div className="flex gap-2">
            <Button variant="outline-default" size="sm">
              Cancel
            </Button>
            <Button 
              variant="primary" 
              size="sm"
              onClick={handleSubmit}
              disabled={comment.trim().length < 3 || isSubmitting}
            >
              {isSubmitting ? 'Posting...' : 'Post Comment'}
            </Button>
          </div>
        </div>
      </CardContent>
    </Card>
  );
}
```

### Support Ticket Form

```tsx
import { useState } from 'react';

function SupportTicketForm() {
  const [ticket, setTicket] = useState({
    subject: '',
    description: '',
    priority: 'medium'
  });
  
  return (
    <Card>
      <CardHeader>
        <h3 className="text-lg font-semibold">Submit Support Ticket</h3>
        <p className="text-sm text-gray-600">
          Describe your issue and we'll help you resolve it
        </p>
      </CardHeader>
      
      <CardContent className="space-y-4">
        <Input 
          label="Subject"
          placeholder="Brief description of the issue..."
          value={ticket.subject}
          onChange={(e) => setTicket({...ticket, subject: e.target.value})}
          required
        />
        
        <div>
          <label className="block text-sm font-medium mb-2">Priority</label>
          <select 
            value={ticket.priority}
            onChange={(e) => setTicket({...ticket, priority: e.target.value})}
            className="w-full border border-gray-300 rounded-md px-3 py-2"
          >
            <option value="low">Low</option>
            <option value="medium">Medium</option>
            <option value="high">High</option>
            <option value="urgent">Urgent</option>
          </select>
        </div>
        
        <Textarea 
          label="Description"
          placeholder="Please provide detailed information about your issue, including steps to reproduce, expected behavior, and any error messages..."
          value={ticket.description}
          onChange={(e) => setTicket({...ticket, description: e.target.value})}
          rows={6}
          maxLength={2000}
          showCharacterCount
          helperText={`${ticket.description.length}/2000 characters`}
          required
        />
      </CardContent>
      
      <CardFooter>
        <div className="flex gap-2 w-full">
          <Button variant="outline-default" className="flex-1">
            Save Draft
          </Button>
          <Button variant="primary" className="flex-1">
            Submit Ticket
          </Button>
        </div>
      </CardFooter>
    </Card>
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Controlled value |
| `defaultValue` | `string` | - | Default uncontrolled value |
| `onChange` | `(e: ChangeEvent) => void` | - | Change handler |
| `placeholder` | `string` | - | Placeholder text |
| `rows` | `number` | `3` | Number of visible rows |
| `autoResize` | `boolean` | `false` | Enable auto-resize functionality |
| `minRows` | `number` | `2` | Minimum rows when auto-resizing |
| `maxRows` | `number` | `10` | Maximum rows when auto-resizing |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size variant |
| `state` | `'default' \| 'success' \| 'error' \| 'warning'` | `'default'` | Validation state |
| `label` | `string` | - | Label text |
| `helperText` | `string` | - | Helper/error text below input |
| `showCharacterCount` | `boolean` | `false` | Show character count |
| `maxLength` | `number` | - | Maximum character limit |
| `minLength` | `number` | - | Minimum character requirement |
| `disabled` | `boolean` | `false` | Disabled state |
| `readOnly` | `boolean` | `false` | Read-only state |
| `required` | `boolean` | `false` | Required field indicator |
| `className` | `string` | - | Additional CSS classes |

## Design Specifications

- **Padding**: 12px horizontal, 8px vertical
- **Border**: 1px solid with rounded corners (6px)
- **Typography**: Inherits from body text
- **Focus**: Primary color border with shadow
- **Resize**: Vertical resize handle (when not auto-resize)

## Accessibility

- **Keyboard Navigation**: Full keyboard support
- **Screen Readers**: Proper labeling and descriptions
- **ARIA Attributes**: Uses appropriate ARIA labels and states
- **Focus Management**: Clear focus indicators and logical tab order
- **Error Handling**: ARIA error descriptions for invalid states

## Best Practices

1. **Clear Labels**: Always provide descriptive labels
2. **Helper Text**: Use helper text for formatting guidelines
3. **Character Limits**: Set reasonable character limits with clear feedback
4. **Auto-resize**: Use auto-resize for better UX when content length varies
5. **Validation**: Provide real-time feedback for form validation
6. **Accessibility**: Ensure proper ARIA labeling and keyboard navigation
