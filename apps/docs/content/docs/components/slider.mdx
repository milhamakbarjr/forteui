# Slider

A versatile range input component for selecting numeric values within a specified range. Supports single and dual handle modes, step controls, custom formatting, and accessibility features.

## Installation

```bash
npm install forteui-core
```

## Import

```tsx
import { Slider } from 'forteui-core';
```

## Basic Usage

```tsx
import { Slider } from 'forteui-core';

function MyComponent() {
  return (
    <Slider 
      min={0}
      max={100}
      defaultValue={50}
      label="Volume"
    />
  );
}
```


## Custom Formatting

### Custom Value Display

```tsx
import { useState } from 'react';

function CustomFormattingExample() {
  const [volume, setVolume] = useState(5);
  const [bandwidth, setBandwidth] = useState(50);
  
  const formatVolume = (value) => {
    if (value === 0) return 'Muted';
    if (value <= 3) return 'Low';
    if (value <= 6) return 'Medium';
    return 'High';
  };
  
  const formatBandwidth = (value) => {
    return value >= 100 ? `${value / 100}GB/s` : `${value}MB/s`;
  };
  
  return (
    <div className="space-y-6">
      <Slider 
        label="Volume Level"
        min={0}
        max={10}
        value={volume}
        onChange={setVolume}
        formatValue={formatVolume}
      />
      
      <Slider 
        label="Bandwidth"
        min={1}
        max={1000}
        value={bandwidth}
        onChange={setBandwidth}
        formatValue={formatBandwidth}
      />
    </div>
  );
}
```

## Real-World Examples

### Video Player Controls

```tsx
import { useState } from 'react';

function VideoPlayerControls() {
  const [currentTime, setCurrentTime] = useState(45);
  const [volume, setVolume] = useState(75);
  const [playbackSpeed, setPlaybackSpeed] = useState(1);
  
  const totalDuration = 180; // 3 minutes
  
  const formatTime = (seconds) => {
    const mins = Math.floor(seconds / 60);
    const secs = seconds % 60;
    return `${mins}:${secs.toString().padStart(2, '0')}`;
  };
  
  return (
    <Card>
      <CardContent className="p-6 space-y-6">
        <div className="text-center">
          <h3 className="font-semibold mb-2">Video Player Controls</h3>
          <div className="bg-gray-100 h-32 rounded-lg flex items-center justify-center">
            Video Player
          </div>
        </div>
        
        <Slider 
          label="Timeline"
          min={0}
          max={totalDuration}
          value={currentTime}
          onChange={setCurrentTime}
          formatValue={(value) => `${formatTime(value)} / ${formatTime(totalDuration)}`}
          color="primary"
        />
        
        <div className="grid grid-cols-2 gap-4">
          <Slider 
            label="Volume"
            min={0}
            max={100}
            value={volume}
            onChange={setVolume}
            suffix="%"
            size="sm"
          />
          
          <Slider 
            label="Speed"
            min={0.25}
            max={2}
            step={0.25}
            value={playbackSpeed}
            onChange={setPlaybackSpeed}
            suffix="x"
            size="sm"
          />
        </div>
      </CardContent>
    </Card>
  );
}
```

### Audio Equalizer

```tsx
import { useState } from 'react';

function AudioEqualizer() {
  const [frequencies, setFrequencies] = useState({
    '60Hz': 0,
    '170Hz': 0,
    '310Hz': 0,
    '600Hz': 0,
    '1kHz': 0,
    '3kHz': 0,
    '6kHz': 0,
    '12kHz': 0,
    '14kHz': 0,
    '16kHz': 0
  });
  
  const updateFrequency = (freq, value) => {
    setFrequencies(prev => ({ ...prev, [freq]: value }));
  };
  
  return (
    <Card>
      <CardHeader>
        <h3 className="font-semibold">Audio Equalizer</h3>
        <p className="text-sm text-gray-600">Adjust frequency levels</p>
      </CardHeader>
      
      <CardContent>
        <div className="grid grid-cols-5 gap-4">
          {Object.entries(frequencies).map(([freq, value]) => (
            <div key={freq} className="text-center">
              <Slider 
                orientation="vertical"
                min={-12}
                max={12}
                value={value}
                onChange={(newValue) => updateFrequency(freq, newValue)}
                height={120}
                suffix="dB"
                size="sm"
              />
              <p className="text-xs mt-2 font-medium">{freq}</p>
            </div>
          ))}
        </div>
        
        <div className="flex justify-center gap-2 mt-6">
          <Button 
            variant="outline-default" 
            size="sm"
            onClick={() => setFrequencies(Object.fromEntries(
              Object.keys(frequencies).map(freq => [freq, 0])
            ))}
          >
            Reset
          </Button>
          <Button variant="primary" size="sm">
            Save Preset
          </Button>
        </div>
      </CardContent>
    </Card>
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number \| [number, number]` | - | Controlled value(s) |
| `defaultValue` | `number \| [number, number]` | - | Default uncontrolled value(s) |
| `onChange` | `(value) => void` | - | Change handler |
| `min` | `number` | `0` | Minimum value |
| `max` | `number` | `100` | Maximum value |
| `step` | `number` | `1` | Step increment |
| `dual` | `boolean` | `false` | Enable dual handle mode |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Slider orientation |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size variant |
| `color` | `'primary' \| 'success' \| 'warning' \| 'error'` | `'primary'` | Color theme |
| `label` | `string` | - | Label text |
| `prefix` | `string` | - | Value prefix (e.g., "$") |
| `suffix` | `string` | - | Value suffix (e.g., "px") |
| `formatValue` | `(value) => string` | - | Custom value formatter |
| `marks` | `Array<{value: number, label: string}>` | - | Mark indicators |
| `disabled` | `boolean` | `false` | Disabled state |
| `height` | `number` | `200` | Height for vertical sliders |
| `className` | `string` | - | Additional CSS classes |

## Design Specifications

### Horizontal Slider
- **Track Height**: Small (4px), Medium (6px), Large (8px)
- **Handle Size**: Small (16px), Medium (20px), Large (24px)
- **Border Radius**: Fully rounded

### Vertical Slider
- **Track Width**: Small (4px), Medium (6px), Large (8px)
- **Handle Size**: Small (16px), Medium (20px), Large (24px)
- **Default Height**: 200px

## Accessibility

- **Keyboard Navigation**: Arrow keys for fine control, Page Up/Down for large steps
- **Screen Readers**: Proper ARIA labels and value announcements
- **Focus Management**: Clear focus indicators on handles
- **Touch Support**: Touch-friendly handle sizes for mobile devices
- **ARIA Attributes**: Uses `role="slider"` with appropriate value attributes

## Best Practices

1. **Clear Labels**: Always provide descriptive labels
2. **Appropriate Steps**: Choose step values that make sense for your use case
3. **Visual Feedback**: Show current values clearly
4. **Range Validation**: Ensure ranges make logical sense
5. **Touch Targets**: Ensure handles are large enough for touch interaction
6. **Performance**: Debounce onChange handlers for expensive operations
