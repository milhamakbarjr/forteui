# Welcome to ForteUI

ForteUI is a comprehensive React component library that helps you build beautiful, accessible, and performant user interfaces.

## Quick Start

Get started with ForteUI in minutes:

```bash
npm install @forteui/core
```

```jsx
import { Button } from '@forteui/core';

function App() {
  return <Button variant="primary">Hello ForteUI!</Button>;
}
```
